-Add firebase in production
| Your Situation                    | Recommendation                                        |
| --------------------------------- | ----------------------------------------------------- |
| Testing between your own machines | **Skip Firebase** — base64 paste is fine              |
| Sharing with non-technical users  | **Add Firebase later** — 7-char codes are much easier |
| Need 100% offline/air-gapped      | **Never add Firebase**                                |
| Production app with general users | **Add Firebase** — industry standard                  |
