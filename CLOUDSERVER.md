When Would You Need a Real Cloud VPS?

| Scenario                            | Action                                     |
| ----------------------------------- | ------------------------------------------ |
| Your ISP changes your IP frequently | Add Dynamic DNS (DuckDNS) or get static IP |
| You move to a location with CGNAT   | Deploy relay to Oracle Cloud free tier     |
| You need 99.9% uptime               | VPS with redundant power/network           |
| You travel with your laptop         | Engine moves with you, relay stays on VPS  |
